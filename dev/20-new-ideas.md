> 設定項目としてアイデアとして他にどのようなものが考えられる？色々広く提案してみて。 ultrathink

⏺ 現在の設定を把握しました。チューナーアプリとして、以下のような設定項目が考えられます：

  ---
  🎵 音程・チューニング関連

  | 設定項目         | 説明                                                        |
  |------------------|-------------------------------------------------------------|
  | 基準周波数 (A4)  | 440Hz（標準）、442Hz（オーケストラ）、415Hz（バロック）など |
  | 移調設定         | Bb管、Eb管など管楽器用の移調表示                            |
  | オクターブ表示   | C4、C5 のようにオクターブ番号を表示するか                   |
  | セント表示の閾値 | ±何セント以内を「合っている」とみなすか（デフォルト±5など） |
  | 平均律/純正律    | 調律方式の選択                                              |

  ---
  📊 表示・UI 関連

  | 設定項目           | 説明                                         |
  |--------------------|----------------------------------------------|
  | グラフの時間幅     | 表示する履歴の長さ（5秒、10秒、30秒など）    |
  | グラフの縦軸範囲   | 表示する周波数/セント範囲                    |
  | 色のカスタマイズ   | 合っている/外れている時の色                  |
  | ダークモード固定   | システム設定に従う / 常にダーク / 常にライト |
  | アニメーション速度 | スムーズ / 即座 / 無効                       |
  | フォントサイズ     | 音名表示のサイズ（小/中/大）                 |
  | コンパクトモード   | 最小限のUI表示（ライブ演奏時用）             |
  | 画面の向きロック   | 横向き固定/縦向き固定/自動                   |

  ---
  🎤 オーディオ入力関連

  | 設定項目                  | 説明                               |
  |---------------------------|------------------------------------|
  | ノイズゲート閾値          | 音量がこれ以下なら検出しない       |
  | 検出アルゴリズム          | WASM / JavaScript / Auto           |
  | サンプルレート            | 44100Hz / 48000Hz など             |
  | バッファサイズ            | レイテンシ vs 安定性のトレードオフ |
  | 入力ゲイン                | マイク入力の増幅                   |
  | ローパス/ハイパスフィルタ | 特定周波数帯域のみ検出             |

  ---
  🔊 音声出力関連

  | 設定項目               | 説明                       |
  |------------------------|----------------------------|
  | リファレンス音を鳴らす | 目標音を出力               |
  | リファレンス音の波形   | サイン波/矩形波/のこぎり波 |
  | リファレンス音の音量   | 出力ボリューム             |
  | メトロノーム機能       | BPM設定、クリック音        |

  ---
  💾 録音・保存関連

  | 設定項目                | 説明                                  |
  |-------------------------|---------------------------------------|
  | 自動録音開始            | 音が検出されたら自動で録音開始        |
  | 無音で自動停止          | 一定時間無音が続いたら録音停止        |
  | 録音品質 (ビットレート) | MP3のビットレート（128/192/320 kbps） |
  | 自動削除期限            | 〇日後に自動削除                      |
  | 保存先の選択            | IndexedDB / ダウンロードフォルダ直接  |
  | ファイル名フォーマット  | カスタム命名規則                      |
  | 最大保存件数            | 容量制限                              |

  ---
  🎸 楽器特化

  | 設定項目               | 説明                                  |
  |------------------------|---------------------------------------|
  | 楽器プリセット         | ギター/ベース/ウクレレ/バイオリンなど |
  | 弦の本数・チューニング | 標準/ドロップD/オープンGなど          |
  | 検出範囲               | 楽器に合わせた周波数帯                |
  | ストロボモード         | 高精度チューニング用                  |

  ---
  ⚡ パフォーマンス・省電力

  | 設定項目         | 説明                           |
  |------------------|--------------------------------|
  | フレームレート   | 30fps / 60fps / 無制限         |
  | 省電力モード     | バックグラウンド時に処理を停止 |
  | 画面スリープ防止 | チューニング中は画面をオン維持 |

  ---
  🌐 アクセシビリティ

  | 設定項目                 | 説明                             |
  |--------------------------|----------------------------------|
  | 音声読み上げ             | 検出した音名を読み上げ           |
  | 振動フィードバック       | 合っている時にバイブ（モバイル） |
  | ハイコントラストモード   | 視認性向上                       |
  | キーボードショートカット | 開始/停止/保存のショートカット   |

  ---
  📱 その他

  | 設定項目                        | 説明                              |
  |---------------------------------|-----------------------------------|
  | 言語設定                        | 日本語 / English                  |
  | 起動時のパネル                  | 最後の状態を復元 / 常にチューナー |
  | データのエクスポート/インポート | 設定と録音をバックアップ          |
  | ショートカットPWA               | ホーム画面追加の案内              |
  | アップデート通知                | 新機能のお知らせ                  |

  ---
  特にユーザー体験に大きく影響しそうなもの:
  1. 基準周波数 - オーケストラ奏者には必須
  2. ノイズゲート - 環境ノイズ対策
  3. 楽器プリセット - ギタリスト向けに便利
  4. 画面スリープ防止 - 練習中に消えると困る
  5. セント表示の閾値 - 「合っている」の判定基準
