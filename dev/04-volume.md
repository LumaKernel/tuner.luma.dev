> Q: デシベル表記がなぜマイナスのみ対応なのかを教えて。

⏺ dBFS（decibels relative to Full Scale） を使用しているためです。

  dBFSとは

  デジタルオーディオでは、信号の振幅は -1.0 〜 +1.0 の範囲で表現されます（Float32Array）。

  - 0 dBFS = フルスケール（振幅 1.0）= デジタル信号の最大値
  - -20 dBFS = 振幅 0.1（最大の1/10）
  - -40 dBFS = 振幅 0.01（最大の1/100）
  - -60 dBFS = 振幅 0.001（最大の1/1000）

  計算式: dB = 20 × log₁₀(振幅)

  なぜマイナスだけ？

  | 振幅 | dBFS   |
  |------|--------|
  | 1.0  | 0 dB   |
  | 0.5  | -6 dB  |
  | 0.1  | -20 dB |
  | 0.01 | -40 dB |

  振幅が1.0（最大）を超えることはデジタル信号ではクリッピング（歪み）を意味します。正常な入力では振幅は常に1.0以下なので、dBFSは常に0以下（マイナス）になります。

  アナログdBとの違い

  アナログの世界（dBu, dBVなど）では基準点が異なり、プラス値も普通に使われます。dBFSはデジタル信号専用の単位で、最大値を基準（0dB）としているため常にマイナスです。

> Q: 通常工事の音がナンデシベルです,とかいうときのやつはなに?

⏺ それは dB SPL（Sound Pressure Level / 音圧レベル） です。

  dB SPL とは

  人間の聴覚の閾値（聞こえる最小の音圧）を基準（0 dB）とした単位です。

  | dB SPL | 例                                 |
  |--------|------------------------------------|
  | 0 dB   | 聴覚の限界（ほぼ無音）             |
  | 20 dB  | 木の葉のそよぎ                     |
  | 40 dB  | 図書館                             |
  | 60 dB  | 普通の会話                         |
  | 80 dB  | 電車内、掃除機                     |
  | 100 dB | 電車のガード下                     |
  | 120 dB | 飛行機のエンジン近く（痛みの閾値） |

  2つのdBの違い

  |        | dB SPL               | dBFS                 |
  |--------|----------------------|----------------------|
  | 基準点 | 人の聴覚閾値 (20μPa) | デジタル信号の最大値 |
  | 範囲   | 0〜140+              | -∞〜0                |
  | 用途   | 環境音、騒音測定     | デジタル録音         |
  | 測定   | マイク + 騒音計      | デジタル信号値       |

  なぜこのアプリはdBFS？

  マイクからの生のデジタル信号値を表示しているためです。dB SPLを表示するには：
  - 校正済みの測定マイク
  - マイクの感度情報
  - 音圧との対応付け

  が必要で、一般的なPCマイクでは正確なdB SPL測定はできません。


dBFSだということがはっきりわかりやすい表記にしよう。
